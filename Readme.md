## Práctica 3 - FTP en Python

### Requerimientos

`python 3`

### ¿Cómo ejecutar?

Primero debemos ejecutar el servidor de la siguiente manera

`cd servidor`

`python3 servidor.py`

Luego en otra terminal abriremos el cliente y haremos lo siguiente

`cd cliente`

`python3 cliente.py`
